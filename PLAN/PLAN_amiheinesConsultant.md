## 1. The ONE Magic Offer (Phase 1)

**Working name:** "Ship Your First On-Chain Game in 60 Days"
**Audience:** junior full-stack devs curious about blockchain.
**Persona ("Garnon"):** 25–32, 2–4 yrs React/Node experience, has played with Claude Code or Cursor, hears Web3 hype, wants a real portfolio piece + faster path into Web3 jobs / side income. Doesn't want to read a 600-page Solidity book.
**Format:** 6-week live cohort, group-coaching (8–12 seats).
**Price:** $5,000 USD.
**Promise (measurable outcome):** at end of 60 days you have a **deployed, playable on-chain game** on Antelope/Vaulta + a portfolio video walking through it.
**Curriculum (4 actionable steps, MAGIC framework):**
1. Spec your game with AI-agent design loops (Claude Code as pair architect).
2. Smart contract: Antelope C++ contract built with Claude Code, deployed to testnet.
3. Frontend: React + chain SDK, AI-accelerated wiring.
4. Ship + portfolio: mainnet/testnet polish + record your demo video.
**Generous payoff anchor (1/10 rule):** avg Web3 dev salary delta for a junior Web2→Web3 transition is $30k–$50k/yr; portfolio piece + cohort credential is the unlock. Charging $5k = ~10% of year-one delta.
**Infinitely scalable?** Group cohort = yes within reason (cap 12/run, multiple runs/year).
**Clear name?** Outcome is in the title.

### Why a cohort (not a self-paced course)
Cohort is in the "Magic" quadrant (high price, easy convert via paid challenge). Self-paced $97–$199 courses are in the "Needy" quadrant — see `FUTURE_MAYBE.md` §A1.

### Why "AI-accelerated"
Two reasons:
1. **Hook for Garnon** — modern junior devs already use Claude Code / Cursor; framing the cohort as AI-accelerated lowers their perceived skill barrier.
2. **Ami's actual workflow** — building Game with Claude Code is faster, AND every session is recordable content. Eat your own dog food on camera.

## 3. Effortless Enrollment

**No 1-on-1 sales calls. No free webinars.**

**Mechanism:** paid 5-day challenge.
- **Ticket:** $97 GA / $297 VIP (live Q&A access).
- **Format:** Mon–Fri, 60–90 min/day live + recording.
- **Outcome:** by Friday each attendee has a tiny on-chain interaction working on testnet (e.g. mint an NFT, simple contract call).
- **Day 5:** pitch the $5k cohort as the natural next step. Cochrane-style: 60-second offer at the end of a transformation week.
- **Target conversion (base case):** 18 paid attendees → 3 cohort sales (= $15k cohort + $2.3k challenge).
- **Stretch:** 30 attendees → 5 cohort sales (= $25k + $2.9k). Bull (existing audience): 8+ sales.
- **Payment plans:** offer 3-pay ($1,750/mo × 3, +5% uplift) alongside lump $5k. Industry rule of thumb: 40-60% of buyers take the plan; payment plans typically lift conversion 20-40%. See `payment_plans.md` for checkout flow.

Run the challenge in **W10** (Jul 06–12, 2026). Cohort #1 starts **W12** (Jul 20–26).

## 4. The 20-hour CEO — Ami's Dream Week

**Hard cap:** 20 hours/week total (build + content + sales + admin). Anything that doesn't fit gets E-A-D'd.

**4 pillars (Cochrane W7):**

| Pillar | Days/Time | What |
|---|---|---|
| Recharge | every morning + Sun off | gym, walk, prayer/meditation, coffee. Start the week with these, not work. |
| Relationship | weekly date day, kids in mornings | block on calendar like income depends on it (it does) |
| Rainmaker | Tue–Wed–Thu | Tue: record long-form video. Wed: ship cohort/challenge work. Thu: build (Game). |
| Reflection | Mon | thinking day — read, journal, plan, no execution |

**Friday:** off-limits for client work; light admin only.

**EAD audit (weekly):** every subtask is on the chopping block unless it's rainmaker work.
- Eliminate first (do it 0 times).
- Then automate (Claude Code / scripts / Listmonk).
- Then delegate (VA only when revenue justifies — no VA in Phase 1).
